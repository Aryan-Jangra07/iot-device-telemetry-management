# IoT Device Backend 🚀

This project is an IoT backend system built using:

- Node.js
- Express.js
- MongoDB
- Swagger API Documentation

## Features
- Device data storage
- REST APIs
- Swagger UI testing
- MongoDB integration

## APIs

### POST /api/device
Save device telemetry data.

### GET /api/device
Fetch all device data.

## Run Locally

```bash
npm install
node server.js
